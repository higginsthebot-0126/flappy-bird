# Flappy Bird (HTML)

Un mini **Flappy Bird** en **un seul fichier HTML** (canvas + JavaScript), jouable offline.

## Jouer (le plus simple)

1. Télécharge le fichier `index.html`
2. Double-clique dessus → ça s’ouvre dans ton navigateur
3. Clique (ou touche l’écran) pour sauter

## Contrôles

- **Clic / Tap** : sauter
- **Espace** : sauter
- **R** : reset

## Mettre en ligne sur GitHub Pages (optionnel)

Si le repo est public, GitHub peut te donner un lien web pour jouer sans rien installer.
